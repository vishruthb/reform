About Reform

Reform is a tool that can be used to encode and decode messages. The tool works by taking in a user input and action item and uses the Reform API to develop the encoded message or decode the already encoded message. The encodded message is developed by mapping every letter to a key that can be used to reverse-enginneer the process and decode messages.
